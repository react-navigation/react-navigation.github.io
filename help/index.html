If you've encountered a bug with React Navigation, please post an issue and be sure to fill out the issue template.

If you believe there is a feature missing, please create a feature request on Canny, or if you're feeling up for the task of proposing an API for the feature, create a discussion.

If you just need some help, try joining the #help-react-native channel on Reactiflux Discord or post a question to StackOverflow. Also be sure to check our troubleshooting page for common issues and how to fix them.
